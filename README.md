# imagenes-farmacias-u
2. Instrucciones para Administración (GitHub)

ESTRUCTURA DEL REPOSITORIO
Las imágenes deben ubicarse en:
https://mmmoure.github.io/imagenes-farmacias-u/gpt_farma_u/
No usar subcarpetas adicionales dentro de esta carpeta.

CONVENCIÓN DE NOMBRES
Documentos:
IDnnn_descripcion.docx
Ej: ID001_alta_cliente_os.docx

Imágenes asociadas:
IDnnn_descripcion.jpg
IDnnn_pantalla_1.jpg
IDnnn_pantalla_2.png

REGLAS
- El prefijo IDnnn_ debe coincidir exactamente.
- Solo minúsculas, guiones bajos y sin tildes.
- Formatos permitidos: jpg, jpeg, png.

CÓMO SUBIR IMÁGENES NUEVAS
1. Ingresar a GitHub → Add file → Upload files.
2. Subir los archivos ya renombrados.
3. Confirmar con Commit.
4. Probar la URL:
https://mmmoure.github.io/imagenes-farmacias-u/gpt_farma_u/NOMBRE.jpg

MANTENIMIENTO
- Eliminar archivos duplicados.
- Verificar imágenes huérfanas.
- Mantener planilla de IDs y procedimientos actualizada.
